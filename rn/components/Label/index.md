# Label

## 标签

设置了几种常用的文字样式和大小，纯展示组件，无事件

| 属性 |   说明   |                   类型                   |  默认值   |
| :--: | :------: | :--------------------------------------: | :-------: |
| size | 字体大小 |      ['xl', 'lg', 'md', 'sm', 'xs']      |   'md'    |
| type |   类型   | ['default', 'title', 'detail', 'danger'] | 'default' |
